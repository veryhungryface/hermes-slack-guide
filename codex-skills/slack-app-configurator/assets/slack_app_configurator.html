SSlack App Configurator
local only

Slack 앱을 CLI manifest로 세팅

Config token, 앱 이름, 권한 체크박스를 받아 manifest를 만들고 Slack CLI로 검증/생성합니다. 토큰은 브라우저와 이 로컬 서버 메모리에만 머무릅니다.

기본 권한은 Hermes Slack guide의 현재 봇/유저 스코프와 같습니다.

환경세팅

이 도구는 로컬 Python 서버가 GUI를 띄우고, Slack CLI가 앱 manifest 검증/생성을 실행합니다.

  1. Slack CLI 설치 확인: slack version
  2. App Configuration Token 준비
  3. 12시간 안에 세팅을 끝내면 Refresh Token은 필요 없습니다.

Windows에서 실행

Windows도 같은 GUI를 씁니다. Python 3와 Slack CLI만 설치되어 있으면 PowerShell에서 바로 열 수 있습니다.

  1. Slack CLI 설치: irm https://downloads.slack-edge.com/slack-cli/install-windows.ps1 | iex
  2. 설치 확인: slack version
  3. GUI 실행: .\scripts\run_windows.ps1

토큰

이 세팅 도구에서는 config token 하나만 씁니다. 토큰이 12시간 후 만료되면 새 token을 발급받아 다시 넣으면 됩니다.

앱정보

소문자 영어로 입력하세요. 필요한 경우 숫자, 점(.), 하이픈(-), 밑줄(_)만 함께 쓸 수 있습니다.
